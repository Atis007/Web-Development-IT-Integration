XHTML

XHTML (Extensible HyperText Markup Language) is a structured hypertext markup language that delegates the visual presentation of web content to CSS (Cascading Style Sheets). It has a modular concept and works in cooperation with XML (Extensible Markup Language) applications.

CSS

The W3C (World Wide Web Consortium) defined CSS (Cascading Style Sheets) as a standard for controlling the presentation and layout of web pages.
